# TaskMenagerApp
Product Backlog:

1. As a user, I want to be able to register for the application by providing my username, email address, and password.
2. As a user, I want to be able to log in to the application using my registered credentials.
3. As a user, I want to be able to log out of the application.
4. As a user, I want to be able to view a list of my tasks.
5. As a user, I want to be able to create a new task.
6. As a user, I want to be able to edit an existing task.
7. As a user, I want to be able to delete a task.
8. As a user, I want to be able to mark a task as completed.
9. As a user, I want to be able to filter tasks by status (completed or not completed).
10. As an admin user, I want to be able to view a list of all users in the application.
11. As an admin user, I want to be able to edit user information (username, email, and password).
